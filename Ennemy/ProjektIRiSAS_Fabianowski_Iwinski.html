Zadowolenie a zarobki

Przypisujemy ocenom zadowolenia z życia wagi od -3 (najmniejsze zadowolenie) do 3 (największe), a następnie tworzymy wykres słupkowy ukazujące średnie zadowolenie w kategoriach zarobkowych (krok co 500zł)

zarobki_zadowolenie = diagnozaOsoby2011[c(2086, 2217)]
zarobki_zadowolenie2 = na.omit(zarobki_zadowolenie)
w = numeric(length(zarobki_zadowolenie2[, 1]))
poziomy = levels(zarobki_zadowolenie2[, 1])
for (i in 2:length(poziomy)) {
    l = zarobki_zadowolenie2[, 1] == poziomy[i]
    l = l * (5 - i)
    p = w + l
    w = p
}


zarobki_cut = cut(zarobki_zadowolenie2[, 2], breaks = c(seq(0, 6000, by = 500), 
    Inf), right = F, include.highest = T)
dane = tapply(w, zarobki_cut, mean)

Obliczanie gęstości zmiennych ilościowych

Tworzymy funkcję liczącą i rysującą jądrowy estymator gestości, dla zmiennych ilościowych w zależności od płci

f <- function(x, plec = "m", przybliznormalnym = F) {
    if (is.numeric(x)) {
        if (plec == "o") {
            d = density(x, na.rm = T)
            y = x
        } else {
            l = diagnozaOsoby2011$PLEC == "mezczyzna"
            if (plec == "m") {
                y = x[l]
                d = density(y, na.rm = T)
            } else {
                l = !l
                y = x[l]
                d = density(y, na.rm = T)
            }
        }
        plot(d, xlab = "x", ylab = "gęstość(x)", main = "gęstość z próbki wybranej cechy")
        if (przybliznormalnym == T) {
            odchylenie = (var(y, na.rm = T))^(1/2)
            oczekiwana = mean(y, na.rm = T)
            cat(paste("odchylenie standardowe = ", odchylenie))
            cat(paste("\nśrednia = ", oczekiwana))
            dol = oczekiwana - 3 * odchylenie
            gora = oczekiwana + 3 * odchylenie
            k = seq(from = dol, to = gora, by = 0.1)
            lines(k, dnorm(k, mean = oczekiwana, sd = odchylenie), col = "red", 
                lty = 2)
            legend(120, 0.02, c("gęstość z próbki", "rozkład normalny"), col = 1:2, 
                lty = 1:2)
        }
    } else {
        cat("zly argument")
    }
}
